#WML

Machine learning tools library.

#Dependencies
- Linux or Mac OS
- Python ≥ 3.6
- torchvision
- OpenCV
- pycocotools
- gcc & g++ ≥ 4.9
- [wtfop](https://github.com/vghost2008/wtfop) custom tensorflow op.
